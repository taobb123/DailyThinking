
### 2025/5/12

思维的本质是什么？古希腊有位哲人说思想是”火“，我非常赞同。

首先，我们在思考的时候运用的是”观念“，所以思维的本质看起来是观念，那么观念的本质是什么？

这是种层层递进的关系，想探索观念的本质，更好的方法是探究观念的终极，终极的观念像什么？沿着这个线索，你就得到了观念是”火“的信念。

从答案来看，火意味着连续的事象，火可以从远方绵延，一直延伸，熊熊大火，变成了火星、变成了烟花、变成了诗、变成了世间种种事象。。。

科学对思维的探究更加严谨，从苏格拉底的三段论 ，笛卡尔的”我思故我在“，到达尔文的物种演化，都给了我们不少启发。

科学界对于思想的主要争论在于：身心合一还是身心分离。主张身心合一的被称作”唯心派”，主张身心分离的被称作“唯物派”。

唯心派宗师有笛卡尔，笛卡尔主张对一切现象加以怀疑,直到意识到一个我是不能再怀疑了，只有我的思维是真实的。

唯物派有休谟，休谟的立场就是相信经验，人的感觉会欺骗自己，所以通往真理的道路只有靠不断观察总结日常的经验。

后来人们发现只要站了队，就必须为自己的学派辩解，最终的目的离科学越来越远，为了平息无益的纷争，出现了调和的学说。

黑格尔的出现就是这样一种标志，人们对于唯心或是唯物听的够多，对于真理还是不得其门而入，黑格尔打破了这种格局，提出了辩证法。

辩证法说事物是互相转化的，世界的基础是矛盾，有个形象的例子：种子埋到土里生根发芽、开花结果，每一次变化都是对自我的否定，矛盾的转化。

辩证法让人们找出矛盾，解决 矛盾，变本加厉，后来在马克思的《资本论》中运用到了社会改革，造成了某种文化的退步。

辩证法一旦让人们接受了矛盾的说法，科学就面临了停滞，因为无论如何指出矛盾存在，矛盾的一方永远可以反对另一方扩大自己的阵营，此时就没有了理性的讨论。

一个学说的没落就带来一个学说的兴起，科学慢慢更加理性，从原来的人们总想一步达到真理，到人们认清了自己的局限，摆脱了乌托邦的梦想。

现在得到科学共识的寻找真理的方法是：假设、演绎、归纳、证明、新的假设。一个理论最重要的是能够证伪，承认真理只是一种概率，没有终极的真理，真理是一个追求的过程。

当今日新月异的AI技术也是得益于这种思想：放弃准确的符号语言，追求一种概率，怎么实现和人类更好地协同。

而协同最重要的是效率和工具，于是思想就只是和外部沟通协作的工具。

写作可以得到心灵平和，当你有东西输出的时候，你是属于你自己的。

人总是被外物推着走，习惯了愤怒、愚蠢，人找不到自己的内心时就会丢了自己，没了心灵等于失去了家园，精神就会遭受流离失所的痛苦。

我们在学校的时候是大量输入、为一场考试输出，但等到了社会需要施展一番的时候，却被这种教育束缚了手脚，因为社会很多有头有脸的人用的方法是“输出倒逼输入”。

怎么理解“输出倒逼输入”？在电脑中有一种说法“黑箱操作”，我们安装的游戏即使一行代码看不懂也能玩得风生水起，这就是电脑的一个黑箱，穿过一大堆晦涩难懂的代码，我们照样可以熟练操控按键完成杀敌的操作。

给你一个函数，即使你不明白底层的逻辑，你依然能输入正确的参数使用这个函数，我们总会持有一种期望，通过输入得到一个满意的输出。

而这个函数在你眼里不再只是单纯的输入，它也包含了你期待的输出，游戏不再是单纯的动画，它也让你得到期待的快感。

函数是一个工具，是你可以随时调用得到结果的输出工具，这个工具让你学会输出倒逼输入，你掌握的就是这样一个模型。

只会输入不一定得到满意的输出，这就像博概率，譬如寒窗苦读的书呆子不分五谷、四体不勤。

但是如果你懂得输出倒逼输入，情况就逆转了，这就相当于你掌握了一个工具，之前的方法只是在模仿一个工具、照猫画虎，因为工具就是一个输入到输出的模型，允许工具存在黑箱，可以操纵黑箱，当然就有了无限可能，这正是输入到输出无法做到的，因为它不能忍受我们无法窥视的“黑箱”。

所以我们要明白打造这种模型才是关键，放弃我们的路径依赖，放弃学校教你的“大量输入为一场考试输出”的习惯，善于创造并使用工具是人和动物的最大的差别。

### 2025/5/16
长时间做一个任务容易疲倦，你或许因为一时的挫折就放弃了，你放弃了就会失去完成挑战的成就感。

那么换一个什么路径比较合适呢？是分解任务变成小步快跑，还是死磕到底最终放弃？

我自认为有效的方法是在多个任务的间隙完成休整。

先明确一点，人类的上班历史其实只有二百多年，人不适合一直工作，人类可能还没有从贪玩中变得理性，让人持续时间工作并不合理，只是现如今人被社会上的责任习惯等因素牵制着。

我们的亲身体验是人在游戏的时候精力充沛，不会遇到什么挫折感。

当然不可能把工作完全变为游戏，但可以通过游戏的作用调动人的情绪，主动暗示自己：工作就是游戏，让工作变得有升级感。

不仅要暗示自己把工作当游戏，还要偶尔放下工作去真正玩几把游戏，这样的切换更容易把游戏未散的余热投入工作，使之变成你充电的技巧。

在工作中感受玩游戏的乐趣，游戏时想着工作的卡点，在这个间隙，你的潜意识就会自动整理有序，等着你更好地完成下一次任务。

这就是在间隙中休息的精要所在。

你不能只钻在一件事里，这会让你的精神由于过度专注而很快涣散，若是有几件事情轮换交替，你的潜意识才能接手部分工作。

所以长时间办公的人强调运动，长时间劳作的人强调休息。

卢梭就在《忏悔录》中提到自己的学习方法——开始的时候他钻研一个问题不能集中精神半个小时，但是让他连续研究几个不同的问题，即使毫不间断，他也能轻松愉快地一个一个地寻思下去，这一问题可以消除另一问题所带来的疲劳，用不着休息一下脑筋。他就在治学计划中充分利用他发现的这一特点，对一些问题交替进行研究，即使整天用功也不觉得疲倦了。以后当他再研究学问还会去田园里做一些零星活计做消遣，每件事都做得很愉快。

你可以根据自己的喜爱来制定你的任务顺序，你可以先玩游戏再工作，也可以先工作再游戏，你还可以最后加上一组运动。

只要注意一点，这几件事情是你喜爱的，要保证过程是你一直享受着的， 你能调整事情的先后顺序来验证怎么组合来实现让你做每件事情都更加纯熟。

最后你会走上一条越做越开心、越开心越做的光明坦途。



### 2025/5/17
我理解的阅读的本质是搜索

阅读中去主动搜索比你被动吸收要好，主动搜索就是不用每个字都阅读，你可以跳着看，当你已经预知了后面的内容，就不需要再逐字斟酌了。

逐字阅读是一个毛病，通篇这样阅读下来，你发现即使你认识所有的字，也未必能复述里面的意思。

搜索阅读有一个好处：你在刻意跳过一些内容的时候会快速汲取有用养分，不会无的放矢。

搜索阅读会让你更主动，更调动你的反思能力，阅读时你会不自觉地想：我为什么要读这些字，它能对我有什么帮助？

这些并不是功利主义的想法，当然也有泛读的方法，像海绵一样广泛吸收，先了解共通的文化理念作为基础，这只是初级的方法，是学跑步前的走路。

如果你放弃了搜索阅读法就失去了精进的机会。我们阅读学习的终极目的一定是形成自己独特的文化属性，不是死记硬背、照本宣科，成为他人思想的传道者。

我们使用搜索阅读首先要搜索，搜索什么？第一步是向内搜索，问内心你想要什么，是对什么感兴趣，对某一个问题有疑惑？或是要验证自己的想法是否正确？

第二步才是搜索文章内容，要看能否找到和你的价值点相符的段落，经过筛选，你提取的内容会更有价值。

搜索阅读的重点是要掌握第一步向内搜索法，不然后面阅读搜索就是无本之源，你也不知道最终要跳到哪里。

向内搜索法的理论依据是：你需要有一 跟线把相关事物串联起来，把零散信息收集整理成一块画布，编织成一块锦缎。

你在观察局部的时候应有大局观，你在构建整幅地图的时候应该有明确的路线。

若是把这跟线命名，就叫做”时间线“，用时间线串联你的过去未来，你可以先让时间线出现在心里，运用你的生活经验，用你对历史的敏感，或运用你对于一个问题发展过程的清晰逻辑。。。用因人而异的方法创造你自己的内在”时间线“。

有了时间线，你能利用它在零散的信息世界建立自己的根据地，打造你的自由世界，把信息变成你构建每座建筑的材料。

各个时间线就像不同的”平行世界“，平行世界能否和现实世界相遇，就在于你是否真正地用好了这个世界的经验材料，是幻想还是真实只有你自己亲自验证，如果你说你已经脱离了地球在光年之外我也不否定，因为地球之内没有证据。



### 2025/5/18
论写作的方法

任何事情都有方法，因为万事是相互关联的，写作只是揭示大千世界纵横阡陌的关系的一角。

把写作看成一幅地图，我们只需要勾勒地图上的地标性建筑就能一窥全貌。

列举几个地标，比如指示牌、分岔口、目的地、路途等等。

指示牌指的是你文章的中心问题，有了问题寻找解决问题的途径，就踏上了路途；各种方法存在矛盾和争论，就遇到了分岔口；等争论结束，你选择一条路坚定方向，最终走完了目的地。

但目的地只是暂时的地点，它不代表着结束，写作是没有终点的，它更像是在一直指路。

写作像地图导航仪，这一次的结束象征着它下一次被打开的起始；写作像用手指月，你看到的不是手指而是循着指向看到的月影星辉。

写作没有起点终点，只有暂时的坐标，写作只是无限宇宙中的渺小的观察者。

所以写作是无极的，但它存在方法，你指向月亮就是好方法，你指向一堆粪就是烂方法。

写作就是找到一块璞玉精心雕琢，而不是找一个钢炉妄想炼金，所以写作有利有弊。

如果你接受的观念让你做出正确的行动就是利，若是接受了邪恶的观念为祸人间就是弊。

所以文章的普世价值也无非在宣扬一方观点的同时打压另一种观点，尽量让好的观念大行其道，扩大好影响的范围，促进人们选择正确的行动。

写作应该有广泛而深远的影响，它不是你随意换身儿新衣服，它像一个密友在夸你衣服很漂亮，你换一次衣服她就对你品头论足，时刻提醒你品味的变化。

好的写作程序是你不仅清晰知道自己表达的事情，而且要把事情有利的一面交代清楚，否则就是冒犯唐突品味变丑了。

在写作表达这个过程中你不断丢弃一些石子再捡起一些更漂亮的，你会因为西瓜而丢弃芝麻，你走向的是越来越丰富的道路不是越来越匮乏，方向走反了就是倒反天罡、贻笑大方。

斯蒂芬.金说过：写作不是人生，但我认为有的时候它是一条重回人生之路。


### 2025/5/19
论阅读的价值

《如何阅读一本书》的作者开始提出一个问题：在学校教育阶段很多人的阅读水平没有提升，大学生只有小学六年纪的阅读水平。

他围绕这个问题拆解了阅读的四个层次：一基础阅读、二检视阅读、三分析阅读、四主题阅读。可惜由于学校在阅读教育的疏忽，导致大学生才掌握了基础阅读。

剩下的就是作者按照每个阅读层次展开说明各自的特征和相互间的递进关系，由于是一本关于阅读的工具书，所以每一个解释过于详细繁琐，我发现了其中一点很有启发意义。

作者在分析阅读中为了讲解如何分类词语以找到重点分析作者的意图中举了一个恰当的比喻：一本书就像是一位老师，你只有清楚课程的类型，是历史科学还是哲学，你才明白老师教的具体知识，然后才找到一个好学生所处的位置。

作者通篇的方法都可以归结为学生和老师的关系，在阅读初级你只能跟着老师学发音拼音，高级一点你会根据课程类型总结出知识大纲，最后你能通过主题阅读去质疑老师的错误提出改进方法。

书中有大量的实例以方便理解每一个步骤的作用和实践，如果略去很多中间的分类讲解，只保留其中的一句话也是很能提升自己的阅读水平。

比如让你思考作者在写书的时候提出的问题和解决的方法，哪些问题是无法解决的，解决方法的类型是实验还是推论或者两者兼备。

以上这两点就足以让人深入阅读的真谛了，一：你把书本当做是你的老师，只是这位老师不会带领你需要你主动去挖掘。二：这位老师向你提出了什么问题，你如何解决这些问题，也就是老师是如何解答的。

重点就是你要带着问题才能找到答案，没有问题再怎么一步步学习如何阅读一本书也是收效甚微。

你没有带着问题去找老师自然不会有知识上的长进，你只有通过和老师的一番热烈讨论或是唇枪舌剑才会有一丝成绩。

问题就像一块冰，你最后看到它消融无形你才能满意，否则你阅读一本书就是前后一致不知所谓没有丝毫改变，除非你当读书只是消遣没有更高的目标否则就是浪费时间。

阅读的价值就是你带着解决问题的心，经过努力接收老师关于这些问题的教学，让问题一点点化解、坚冰消融变成了液态。

问题和对于老师的了解是阅读不可缺少的价值要素，关于对老师的了解如何更多，《如何阅读一本书》有十分详细的分解，例如如何分类主题、怎么做笔记、怎么用一句话总结整本书的故事。不是专业的文字工作者不读也罢。



### 2025/5/20
这世界是个巨大的交易市场，吃穿住行、活动娱乐、贩卖焦虑、制造焦虑。

有时候焦虑不是谁能制造的，人本质上就会焦虑，这是人性共通的底色，只是有人发明了解决焦虑的办法，支个摊一吆喝就能吸引别人的注意。

隐藏的商机往往就在你自己身上，你把自己放在同类中想想你和别人的共同焦虑，说不定就发现了机遇，你发现了大众的焦虑，提出解决方案，你就赢得了市场。

最好你进入市场前就摸清了同行的动向，若是你没有别人便宜，没有好的获客渠道，凭一腔热血会面临惨淡收场的危险。

巴菲特说好的生意要有自己的护城河，这里包含的真理就是你越擅长做的越好，做的越好就越擅长，形成飞轮效应。

你做的好才能形成护城河，做的不好当然鲜有人买账，这个道理妇孺皆知，可是你为什么依然做不好？

有人可能会怪罪成长环境，没有出生在市场旁得到耳濡目染，别人在寻找商机时你只能玩泥巴，但是你能把泥巴玩的出彩，也足够得到市场的反馈了。

所以环境是次要的，关键看你个人的心境是什么样的，你是利己的心就会被市场淘汰，你保持利他的心就会留在市场。

即使这个利他是为了利己，那也是你在给予别人好处之后才能获得，而且利己不可能消失，只是别让利己超过了一定的限度，别让利己心变成害他之心。

每个人的自私才促进了市场的繁荣，大公无私并不是理想世界，因为只听命于一个统一的命令，必定会偏离每个独立的私心，反而成了最大的不公平。

人为了自己的需求创造了各异的焦虑，这明显会促进市场的兴起，会激起个人交易的热情，饮食男女灯红酒绿热气腾腾，市场永恒的格局，万古不变的市井图。

是时候扪心自问了，你的优势是什么？你能在哪里挖一道护城河？不然就找找你的焦虑，问问别人有没有你的焦虑，找到焦虑就尝试着让自己熟练解决焦虑教给别人。

要么做自己的人生导师，要么做别人的人生导师，舍此没有第二条路，你要记得市场上的人可是很自私哦，他那直勾勾的眼睛在你身上搜寻着干粮要满足自己的饥渴，在那之前他才不会施舍你一个铜币嘞！


### 2025/5/21
论敌人的重要性

万历十五年海倭横行，倭寇的海军远超明朝政府的装备，他们自日本沿岛而来，其中夹杂很多汉人，即使什么人都有，这帮杂牌军面对明朝海军依然如入无人之境，只要上了岸把附近民居扫劫一空便全身而退。

万历皇帝派大将胡宗宪前去剿灭倭寇，胡深知双方战力差距于是设计用计谋捉拿海倭，他派人打探到倭寇的诉求，原来明朝一直不肯开放海岸阻止人们做外贸，为了生计，他们只好做出一些动静想要引起政府的注意换得口岸开放，胡宗宪利用他们的心理先安抚假装请他们友好谈判，等大部分倭寇放松了警惕以为达到了目的，胡宗宪趁机让埋伏的士兵把倭寇全部清理了。

可想而知后来倭寇面对狡诈的政府更变本加厉了，这些部队配合默契，以某种口哨为令，有先锋有侧翼支援，明军面对他们就是一盘散沙，危难之际，首辅张居正提携戚继光掌握兵权，戚继光提出要重新整顿军队的构成，制定了详细的计划，比如需要从哪里征兵，哪些人不能被应征，优先征取来自农村的精壮小伙，因为农民更加坚强容易遵守上官的命令，相反来自城市的人总考虑自己眼神闪烁一副聪明相，就算逃跑的时候也要撺掇周围的人一起乱纪，只因一旦被抓到可以把责任推到言语木讷的那个人身上。

两年后戚继光打造了军纪严明的戚家军，经过战术的改良和精神体魄的严格操练，这支戚家军犹如敢死队很快就把倭寇之患解决了，戚家军出了名的纪律严格，戚继光也以身作则亲自出征，为了规范军纪曾杀死过自己的二儿子，对军队也是多加鼓励，劝他们多想想家里的亲人辛苦种田给政府上交田赋税，你们在军队平时养尊处优不用面朝黄土背朝天，只需要用人之时你能上阵杀敌，况且你平日勤加操练上战场是不用恐慌的，军人的信心、尊严是最宝贵的，只有胜利才能增强信心，唯有刻苦磨练平时的技术才有胜利的希望，保持军人的尊严。

戚家军平息倭寇，驱逐鞑虏，所向披靡令敌闻风丧胆，维护了海上和边境的和平，戚继光的军事才能也被载入史册。在万历那个文官当道的年代，武将被作为工具召之即来挥之即去，武官没有发言权，武将在前线牺牲报国还要被一帮文人反讥为鲁莽暴虐，戚继光在武官中不太一样，他精通兵法，发明出鸳鸯阵法，就是十二个人组成一个方阵，前面两个拿盾牌匍匐前进用手中的兵器扫荡马腿，其后是狼筅兵举着长矛等待斩杀落马之敌，两侧是防守的，后面的人兵器又不同，有鸟铳有长枪负责进攻或者转为掩护。戚继光擅长写诗喜欢和文官一起饮酒作对，也喜欢跑到危险的前线勘探敌情颇为自己的矫健身法感到自豪。

在那个明朝政府臃肿无力回天的年间，若不是张居正慧眼识才，万历可能就少了一位大将，若不是内忧外患的年代，戚继光的军事才能也不能得到发挥，戚继光面对帝国倾覆前夕的各种敌人，贡献出了自己的光和热，敌人越强大，戚继光才在军神的道路上越坚定。
